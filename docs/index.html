Find Your Session ID

The session ID authenticates requests to AirGarage on your behalf. You grab it once from your browser.

  1. Open pay.airgarage.com in Chrome and log in to your account
  2. Open DevTools: press F12 or right-click → Inspect
  3. Go to the Application tab → Cookies → https://pay.airgarage.com
  4. Find the cookie named sessionid and copy its value
Heads up: Session IDs can expire when you log out or after extended inactivity. If the automation stops working, repeat this step to grab a fresh one.

Find Your Spot ID

Each AirGarage location has a unique numeric spot ID. You'll find it in the network request when you start a session.

  1. Go to pay.airgarage.com and navigate to your parking location
  2. Open DevTools → Network tab. Check Preserve log
  3. Start a parking session (don't worry — you can cancel it immediately)
  4. In the Network tab, find the POST request to /api/slots
  5. Click it → Payload tab → look for "spot": XXXX — that number is your spot ID
Tip: Your spot ID is tied to the specific parking lot, not your account. Everyone at the same location uses the same spot ID.

GitHub Setup

Fork the repo and add these secrets. The workflow reads them at runtime — nothing sensitive in your code.

  1. Fork the repo: github.com/kyukim-sf/AirGarage → click Fork (top right)
  2. In your fork, go to Settings → Secrets and variables → Actions
  3. Click New repository secret and add each secret below
Secrets to Add
Enable Actions: After forking, go to the Actions tab in your repo and click "I understand my workflows, go ahead and enable them."
05

cron-job.org Setup

This triggers your GitHub workflow at the exact right time, bypassing GitHub's unreliable scheduler.

  1. Create a free account at console.cron-job.org
  2. Create a GitHub Personal Access Token with only the workflow scope at github.com/settings/tokens/new
  3. Create two cron jobs using the settings below
Job 1 — Start Parking
Job 2 — End Parking
For both jobs: Set Request Method to POST, add headers Authorization: Bearer YOUR_TOKEN, Accept: application/vnd.github.v3+json, and Content-Type: application/json. Timezone: your local timezone.
✓

You're all set!

Your parking will now start and end automatically every weekday. Check your GitHub Actions tab after the first scheduled run to confirm everything worked.

Test it now: Go to GitHub Actions → AirGarage Parking Automation → Run workflow → choose start → confirm it works in the logs. Then run end to close the session.
